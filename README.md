# Sharpie
*Sharpie* is **a IRC Bot built in C#**, compatible with both .NET and Mono.

### Commands
* **#consay [text]** *Consay*
  * Prints [text] to the console.
* **#hello** *HelloWorld*
  * Prints "Hello, World!" to IRC.
* **#version** *Version*
  * Prints *Sharpie* version, OS, and compiler (i.e. .NET or Mono).
